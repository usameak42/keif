    2. Create `brewos-engine/brewos/solvers/percolation.py` with `solve_accurate()`:

       Signature: `def solve_accurate(inp: SimulationInput, method_defaults: dict = None) -> SimulationOutput`

       The `method_defaults` dict provides geometry/flow defaults (bed_depth_m, bed_diameter_m, pressure_bar, porosity, brew_ratio_min, brew_ratio_max). If None, use V60-like defaults for direct testing.

       Internal defaults (used when method_defaults is None):
       ```python
       PERCOLATION_DEFAULTS = {
           "bed_depth_m": 0.050, "bed_diameter_m": 0.080,
           "pressure_bar": 0.0, "porosity": 0.40,
           "brew_ratio_min": 15.0, "brew_ratio_max": 17.0,
       }
       ```

       Implementation steps inside solve_accurate():

       a. Merge method_defaults with PERCOLATION_DEFAULTS (method_defaults overrides).
       b. Resolve PSD: `psd_points, grind_size_um = resolve_psd(inp)` from output_helpers.
       c. Get params: `p = derive_percolation_params(inp.coffee_dose, inp.water_amount, inp.water_temp, grind_size_um, bed_depth_m=defaults["bed_depth_m"], pressure_bar=defaults.get("pressure_bar", inp.pressure_bar), porosity=defaults["porosity"])`.
       d. Set up MOL state vector: `N = p["N"]` (30 nodes), `y0 = np.zeros(3 * N)`. Initial conditions: `y0[N:2*N] = p["c_v0"]` (c_v = gamma_1 * c_sat), `y0[2*N:3*N] = p["psi_s0"]` (psi_s = 1.0).
       e. Apply CO2 bloom modifier if bean_age_days is set (reuse `co2_bloom_factor` from `brewos.utils.co2_bloom`): create a `bloom_fn(t)` that returns a multiplicative factor on kB, exactly as in immersion.py.
       f. Define `percolation_ode(t, y)`:
          - Unpack: `c_h = np.clip(y[0:N], 0.0, p["c_sat"])`, `c_v = np.clip(y[N:2*N], 0.0, p["c_sat"])`, `psi_s = np.clip(y[2*N:3*N], 0.0, 1.0)`
          - Upwind advection: `dc_h_dz = np.zeros(N)`, `dc_h_dz[1:] = (c_h[1:] - c_h[:-1]) / p["dz"]`
          - kB_eff = `p["kB"] * bloom_fn(t)` (or just p["kB"] if no bloom)
          - Extraction kinetics (vectorized, per-node):
            `dc_h_dt = -p["v_darcy"] * dc_h_dz - p["kA"] * (c_h - c_v) + kB_eff * (p["c_sat"] - c_h) * psi_s`
            `dc_v_dt = p["kC"] * (c_h - c_v)`
            `dpsi_s_dt = -p["kD"] * (p["c_sat"] - c_h) * psi_s`
          - Boundary condition: `dc_h_dt[0] = 0.0` (inlet: fresh water, c_h stays 0)
          - Return: `np.concatenate([dc_h_dt, dc_v_dt, dpsi_s_dt])`
       g. Solve: `sol = solve_ivp(percolation_ode, (0.0, inp.brew_time), y0, method='Radau', rtol=1e-6, atol=1e-8, t_eval=np.linspace(0, inp.brew_time, 100))`
       h. If `sol.success is False`: raise `RuntimeError(f"Percolation ODE solve failed: {sol.message}")`
       i. Post-process: For each time step, compute outlet concentration = `c_h_outlet = sol.y[N-1, :]` (last spatial node). Compute EY(t) using Liang scaling:
          ```python
          brew_ratio = inp.water_amount / inp.coffee_dose
          for i, t_val in enumerate(sol.t):
              c_outlet_i = sol.y[N-1, i]  # outlet c_h at time t
              c_outlet_i = max(0.0, min(c_outlet_i, p["c_sat"]))
              # Average c_h across all nodes at time t for TDS
              c_h_avg = np.mean(np.clip(sol.y[0:N, i], 0.0, p["c_sat"]))
              # TDS from outlet stream concentration
              tds_i = c_h_avg / (rho_w + c_h_avg) * 100.0
              # EY from mass balance
              ey_raw = (c_h_avg * (inp.water_amount / 1000.0)) / (inp.coffee_dose / 1000.0) / 10.0
              ey_scaled = ey_raw * K_liang / E_max  # Liang 2021 scaling
              extraction_curve.append(ExtractionPoint(t=round(t_val, 2), ey=round(ey_scaled, 3)))
          ```
