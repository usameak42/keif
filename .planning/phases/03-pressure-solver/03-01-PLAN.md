      ```python
      PRESSURE_DEFAULTS = {
          "bed_depth_m": 0.005,       # 5mm moka pot filter basket
          "bed_diameter_m": 0.070,    # 3-cup Bialetti
          "porosity": 0.38,
          "method_type": "moka",
          "ey_target_pct": 18.0,
          "Q_heater_W": 800.0,        # medium stove burner
          "m_water_g": 150.0,         # 3-cup fill
          "h_loss": 10.0,             # W/(m^2*K) aluminum pot
          "A_surface_m2": 0.020,      # external surface area
          "T_ambient_C": 22.0,
      }
      ```

    Helper functions:
    - `steam_pressure_pa(T_celsius)` -- Clausius-Clapeyron: `P_REF * np.exp(L_V / R_GAS * (1.0/T_REF - 1.0/(T_celsius+273.15)))`
    - `water_viscosity(T_celsius)` -- Arrhenius-like: `MU_REF * np.exp(B_MU * (1.0/(T_celsius+273.15) - 1.0/T_MU_REF))`

    `solve_accurate(inp, method_defaults=None)`:
    1. Merge PRESSURE_DEFAULTS with method_defaults
    2. `resolve_psd(inp)` for grind_size_um and psd_curve
    3. `derive_immersion_params()` for Moroney rate coefficients (kA, kB, kC, kD, c_sat)
    4. `kozeny_carman_permeability()` for K_bed from grind_size and porosity
    5. Extract thermal params from defaults: Q_heater_W, m_water_g, h_loss, A_surface_m2, T_ambient_C
    6. Compute bed area: `A_bed = np.pi * (defaults["bed_diameter_m"]/2)**2`
    7. Compute V_bed_pore: `A_bed * defaults["bed_depth_m"] * defaults["porosity"]`
    8. Initial conditions: `y0 = [T_celsius, 0.0, 0.0, c_v0, psi_s0, 0.0]` (T, V_ext, c_h, c_v, psi_s, M_cup)
    9. CO2 bloom modifier (same pattern as immersion.py)
    10. Define `moka_pot_ode(t, y, ...)` with 6 state variables:
        - `T, V_ext, c_h, c_v, psi_s, M_cup = y`
        - `m_w = max(m_w0_kg - rho_w * V_ext, 0.01)` (remaining water mass in kg)
        - `P_steam = steam_pressure_pa(T)` and `dp_net = max(P_steam - P_REF, 0.0)`
        - `mu = water_viscosity(T)`
        - Darcy flow: `q = K_bed / mu * dp_net / L_bed` capped at 5e-3 m/s
        - `dT = (Q - h_loss * A_surface * (T - T_ambient)) / (m_w * C_PW)`
        - `dV = q * A_bed`
        - Moroney extraction with advective loss: `dc_h = -kA*(c_h - c_v) + kB_eff*(c_sat - c_h)*psi_s - (q/max(V_bed_pore, 1e-9))*c_h`
        - `dc_v = kC * (c_h - c_v)`
        - `dpsi_s = -kD * (c_sat - c_h) * psi_s`
        - `dM_cup = q * A_bed * c_h`
        - Bound clip all state vars inside ODE (SOLV-08)
    11. Termination event: `water_exhausted(t, y)` returns `m_w0_kg - rho_w * y[1] - 0.05*m_w0_kg`, terminal=True, direction=-1
    12. `solve_ivp(moka_pot_ode, (0, brew_time), y0, method='Radau', t_eval=linspace(0, brew_time, 100), rtol=1e-6, atol=1e-8, events=[water_exhausted])`
    13. Post-process: extract c_h(t) from sol.y[2], compute EY_raw, scale to ey_target_pct (Liang-style scaling), build extraction_curve
    14. Return SimulationOutput with all 7 fields via shared helpers

    `solve_fast(inp, method_defaults=None)`:
    1. Merge defaults, resolve PSD
    2. R_brew = water_amount / coffee_dose
    3. EY_eq = ey_target_pct from defaults (18.0% for moka)
    4. Biexponential: `EY_t = EY_eq * (1 - A1_MOKA*exp(-t/TAU1_MOKA) - (1-A1_MOKA)*exp(-t/TAU2_MOKA))`
    5. Build SimulationOutput with shared helpers, mode_used="fast"

    ```python
    """Moka Pot -- pressure solver with stovetop geometry defaults."""

    from brewos.models.inputs import SimulationInput
    from brewos.models.outputs import SimulationOutput
    from brewos.solvers.pressure import solve_accurate, solve_fast

    MOKA_POT_DEFAULTS = {
        "bed_depth_m": 0.005,       # 5mm moka pot filter basket
        "bed_diameter_m": 0.070,    # 3-cup Bialetti
        "porosity": 0.38,
        "method_type": "moka",
        "ey_target_pct": 18.0,     # mid-range moka EY
        "Q_heater_W": 800.0,       # medium stove burner
        "m_water_g": 150.0,        # 3-cup fill
        "h_loss": 10.0,            # W/(m^2*K) aluminum pot
        "A_surface_m2": 0.020,     # external surface area
        "T_ambient_C": 22.0,
    }

    def simulate(inp: SimulationInput) -> SimulationOutput:
        """Run Moka Pot simulation, dispatching to correct solver mode."""
        if inp.mode.value == "accurate":
            return solve_accurate(inp, method_defaults=MOKA_POT_DEFAULTS)
        else:
            return solve_fast(inp, method_defaults=MOKA_POT_DEFAULTS)
    ```
