**Database initialization (per D-08, D-09):**
- Database file: `"keif-runs.db"`.
- Open with `SQLite.openDatabaseAsync("keif-runs.db")` in a `useRef` to hold the db instance.
- On first hook call, run `CREATE TABLE IF NOT EXISTS saved_runs (id INTEGER PRIMARY KEY AUTOINCREMENT, name TEXT NOT NULL, method TEXT NOT NULL, created_at TEXT NOT NULL, input_json TEXT NOT NULL, output_json TEXT NOT NULL, archived INTEGER NOT NULL DEFAULT 0)`.
- Use a `useEffect` with an async IIFE to open and init. Set an `initialized` ref to `true` after init. Store loading/error state.

**Operations to implement:**

`save(name: string, input: SimulationInput, output: SimulationOutput): Promise<number>`
- INSERT INTO saved_runs (name, method, created_at, input_json, output_json, archived)
- VALUES (name, input.method, new Date().toISOString(), JSON.stringify(input), JSON.stringify(output), 0)
- Return the inserted row id. After save, call reload().

`listActive(): Promise<SavedRun[]>`
- SELECT * FROM saved_runs WHERE archived = 0 ORDER BY created_at DESC

`deleteById(id: number): Promise<void>` (hard delete — per D-10)
- DELETE FROM saved_runs WHERE id = ?
- After delete, call reload().

`archiveOlderThan(days: number): Promise<void>` (soft delete — per D-10)
- Compute cutoff: `new Date(Date.now() - days * 24 * 60 * 60 * 1000).toISOString()`
- UPDATE saved_runs SET archived = 1 WHERE created_at < cutoff AND archived = 0
- After archive, call reload().

`reload()`: re-runs `listActive()`, updates `runs` state, updates `count` state.

Return: `{ runs: SavedRun[], count: number, loading: boolean, error: string | null, save, deleteById, archiveOlderThan, reload }`.

Handle errors in all operations: catch and set `error` state with a descriptive string.
