    4. Set `t_end = inp.brew_time`, `t_eval = np.linspace(0, t_end, max(100, int(t_end)))` (1-second resolution up to brew_time, minimum 100 points)
    5. Call `solve_ivp(moroney_ode, (0.0, t_end), [c_h0, c_v0, psi_s0], method='Radau', t_eval=t_eval, rtol=1e-8, atol=1e-10)`

    9. Compute `TDS_pct = c_h_scaled / rho_w * 100.0`, `R_brew = inp.water_amount / inp.coffee_dose`, `EY_pct = TDS_pct * R_brew`
    10. Build extraction_curve: `[ExtractionPoint(t=float(t), ey=float(ey)) for t, ey in zip(sol.t, EY_pct)]`
