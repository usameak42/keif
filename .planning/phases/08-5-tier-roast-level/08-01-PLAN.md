**1b. Recalibrate CO2_PARAMS** in `brewos-engine/brewos/utils/co2_bloom.py` (per D-02, D-03, D-04):

Replace the entire CO2_PARAMS dict with 5 entries. Derivation methodology:

**Tau values (D-02):** Use Smrke 2018 Table 1 Weibull lambda averages across roast speeds, converted to biexponential tau. Conversion: `tau_biexp_seconds = lambda_hours * 3600`. Split: 20% fast / 80% slow (confirmed from dark-only biexponential fit in Smrke 2018).

Weibull lambda averages from Obsidian data:
- Light: avg(310, 377, 326) = 337.7h
- Medium: avg(190, 232, 270) = 230.7h
- Dark: avg(153, 190, 242) = 195.0h
- Medium-Light: interpolated = (337.7 + 230.7) / 2 = 284.2h
- Medium-Dark: interpolated = (230.7 + 195.0) / 2 = 212.8h

Fast tau (20% of total degassing timescale): tau_fast = lambda_avg * 3600 * 0.20
Slow tau (80%): tau_slow = lambda_avg * 3600 * 0.80

Computed values:
- Light:        tau_fast = 337.7 * 3600 * 0.20 = 243,144s (~2.81d), tau_slow = 337.7 * 3600 * 0.80 = 972,576s (~11.25d)
- Medium-Light: tau_fast = 284.2 * 3600 * 0.20 = 204,624s (~2.37d), tau_slow = 284.2 * 3600 * 0.80 = 818,496s (~9.47d)
- Medium:       tau_fast = 230.7 * 3600 * 0.20 = 166,104s (~1.92d), tau_slow = 230.7 * 3600 * 0.80 = 664,416s (~7.69d)
- Medium-Dark:  tau_fast = 212.8 * 3600 * 0.20 = 153,216s (~1.77d), tau_slow = 212.8 * 3600 * 0.80 = 612,864s (~7.09d)
- Dark:         tau_fast = 195.0 * 3600 * 0.20 = 140,400s (~1.63d), tau_slow = 195.0 * 3600 * 0.80 = 561,600s (~6.50d)

A_fast/A_slow fractions: 0.20 / 0.80 for all tiers (from the only confirmed biexponential fit — dark fast roast). Keep uniform since no per-tier biexponential amplitude data exists.

**Beta values (D-03):** Proportional to CO2 content midpoints from Obsidian data. Use linear scaling anchored at medium = 0.20 (current production value):
- CO2 midpoints: light=3.555, medium_light=4.75, medium=6.525, medium_dark=9.00, dark=12.89
- Scale factor: 0.20 / 6.525 = 0.03065
- light: 3.555 * 0.03065 = 0.109 -> round to 0.11
- medium_light: 4.75 * 0.03065 = 0.146 -> round to 0.15
- medium: 6.525 * 0.03065 = 0.200
- medium_dark: 9.00 * 0.03065 = 0.276 -> round to 0.28
- dark: 12.89 * 0.03065 = 0.395 -> round to 0.40
