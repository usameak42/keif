*Priority: SECOND — biggest technical differentiator, removes backend dependency*

- [ ] Port immersion solver to JS (3 ODE, RK45) — French Press + AeroPress-steep

- [ ] Make backend optional: online → backend, offline → on-device
- [ ] Offline mode UI indicator
